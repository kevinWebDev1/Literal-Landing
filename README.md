# 🗝️ Literal - The First Accountability App with a Real Keyholder

[![Coming Soon](https://img.shields.io/badge/Status-Coming%20Soon-blueviolet?style=for-the-badge)](https://literal-landing.vercel.app/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg?style=for-the-badge)](https://opensource.org/licenses/MIT)

**Literal** is the first accountability application that pairs you with a dedicated accountability partner—a real keyholder—to lock your distractions and help you unlock your true potential.

Stop relying on willpower alone. Get matched with someone who holds you to your commitments.

---

## ✨ Features

- **Real Human Accountability**: Connect with a dedicated partner, not just a bot.
- **Goal-Oriented Locking**: Strategic focus sessions managed by your keyholder.
- **Glassmorphism Design**: A premium, modern interface designed for focus.
- **Waitlist Integration**: Seamless signup flow for early access notifications.

## 🛠️ Tech Stack

- **Core**: HTML5, CSS3, JavaScript (Vanilla)
- **Design**: Premium glassmorphism effects, custom HSL color palettes.
- **Backend**: Google Apps Script (for waitlist submissions to Google Sheets).
- **Fonts**: [Outfit](https://fonts.google.com/specimen/Outfit) via Google Fonts.

## 🚀 Getting Started

### Prerequisites

- A modern web browser.
- Git (for cloning and version control).

### Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/kevinWebDev1/Literal-Landing.git
   cd Literal-Landing
   ```

2. **Open the project**:
   Simply open `index.html` in your favorite browser.

## 🎨 Design Philosophy

Literal uses a "Soft Glass" aesthetic:
- **Floating Blobs**: Animated background elements for depth.
- **Noise Overlays**: Subtle texture for a premium feel.
- **Micro-animations**: Pulse effects and smooth transitions to keep the interface feeling alive.
